HTML

    Questions and Answers
  1. What is HTML and why is it important?
    • HTML stands for Hypertext Mark Language and it is just a language where browsers
      can read the text and turn it into a website. HTML format is the use of tags , pictures,
      and whatever else you see on a website. It is important because if HTML was not
      created,then there will be no websites or pages.
  2. What is CSS and why is it important?
    • CSS stands for Cascading Style Sheets. CSS is important because it controls how you
      website or webpage looks like.
  3. How do tags work?
    • Tags are everything you do on a webpage. They work by wrapping around certain
      texts, and apply its function to what it wraps around.
  4. What are attributes and how do they work?
    • Attributes are aditional information about HTML elements. They come as
      name="value" and are in the tags. They provide special things like changing the color
      of the font,changing size, adding additional formats, etc.
  5. Description of at least 4 different tags
    • < p > tag: The "p" tag stands for a paragraph and you use the "p" tag when you want to
      write text for a paragraph.
    • < br > tag: The "br" tag is used for line break, so like starting a new line. The "br" tag is
      used when you want to start a new line for your text, but don't want to use a margin or
      padding for it.
    • < b > tag: The "b" tag stands for bolding words and basically it is use to bold words,
      texts, titles, etc.
    • < li > tag: The "li" tag stands for listing. The "li" tag is usually goes after a "ol" tag or a
      "ul" tag and it is used for listing items, texts,etc.
