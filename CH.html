Hardware

1. What is a motherboard and what does it do?

A motherboard is the largest card in the computer and it basically contains
everything that is needed to be a computer. If there is no motherboard
then there wouldn't even be a computer. The motherboard contains
CPUs, memory, disks drives, video cards and other peripherals. There
are many different types of motherboards and it is used for different types
devices.

2. What is a CPU and what does it do?

The CPU of a computer is the brains of the computer. CPU stands for
Central Processing Unit. The CPU tells the computer what to do when
it is given instructions. The order of its operation is fetch, then decode,
and finally execute. During the fetch stage, the CPU gets the information
and then it transfers to decoding. Decodingis when the information is
broken down into steps, so the computer can understand what to do.
Finally the computer executes the order and preforms the task.

3. What is RAM and what does it do?

RAM stands for random access memory and RAM is basically the
memory of the computer. It is considered as "random access" because
you can access any memory if you know the row and column.






4. What is a harddrive and what does it do?

A hard drive is the computer's main storage media device that stores
all the data on the computer. Most of the computer hard drives are
connected with the motherboard using either a ATA, SCSI, or SATA
cable and a power cable.




5. What is a power supply and what does it do?

The power supply is the AC adapter and what it does is it gives power
to the computer, so the computer can turn on and start functioning. If
there isn't the AC adapter then the computer cant turn on or work unless
there is a different power supply.




6. How do the different components communicate with one another?

The way the different components communicate with each other is through
the motherboard. The motherboard allows each other parts to have power
then able to communicate and work together to make up the computer.

7. What are the keyboard, mouse, and monitor and what do they do?

The keyboard is what lets you type and give the computer commands. The
mouse is what lets you navigate the computer and used to click things.
Finally the monitor is used to display what is going on in the computer.
